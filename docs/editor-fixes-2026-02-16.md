# HWPX Editor Bug Fixes (2026-02-16)

## 반영 항목
- 폰트 목록에서 유료/제한 폰트 항목을 제거하고 기본 폰트를 `Noto Sans KR`로 통일
- 단락 자동 포커스가 표 셀 선택 동작을 가로채는 문제 수정
- 표 편집 동작 보강
  - 셀에서 `Enter`/`Tab` 등으로 인한 선택 분기 정합성 개선
  - 셀에서 `Ctrl/Cmd + Enter`로 아래 문단 삽입
  - 셀에서 `Esc`로 테이블 오브젝트 선택
  - 테이블 선택 상태에서 `Enter`로 아래 문단 삽입
  - `Ctrl/Cmd + Backspace/Delete`로 테이블 삭제
- 셀 글자 스타일 적용 로직 견고화(식별자 호환성 대응)
- 글꼴/색상/테두리/배경 패널 입력을 공용 `ColorPicker` 컴포넌트로 통일
- 페이지 번호 토글 상태를 현재 문단/섹션 설정값과 동기화
- 이탤릭/볼드 렌더 가독성 보강

## 테스트
- 브라우저 기반 동작 확인 완료 (`Playwright`)
- `npm/pnpm` UI 루프 테스트 및 셀/표/서식 흐름 검증 수행
